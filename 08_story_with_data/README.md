# Исследования рынка общепита в Москве для принятия решения об открытии нового заведения

## Задача
Исследование рынка общественного питания на основе открытых данных, подготовка презентации для инвесторов

## Описание данных
Таблица `rest_data`:
- `object_name` — название объекта общественного питания;
- `chain` — сетевой ресторан;
- `object_type` — тип объекта общественного питания;
- `address` — адрес;
- `number` — количество посадочных мест.

## Структура проекта
- Импорт библиотек и загрузка данных
- Предобработка данных
- Исследовательский анализ данных

## Итоговый вывод
Итак, результирующие комментарии:
- кафе, столовая, ресторан - ТОП3 типов заведений по Москве. Вместе они занимают 70% всего рынка общественного питания столицы. 
- 80% всех заведений Москвы это несетевых заведения общественного питания. 
- любимый тип заведений для сетевых представителей это - кафе, предприятие быстрого обслуживания, ресторан. Вместе эти форматы занимают 91% сетевых заведений столицы. 
- стоит отметить, что для сетевых заведений (да и для несетевых) «любимые» размеры по вместимости это 50 мест, 100 мет, 0 мест. 
- если оценивать заведения по усредненному количеству посадочных мест, то лидерами будут: столовая, ресторан, буфет.
- ЦАО, ЮЗАО, САО, ЮАО, ЗАО - округа Москвы в которых есть улицы из ТОП10 по количеству мест общественного питания.
- Для заведений на улицах с количеством мест общественного питания более 50 есть ярко выраженный формат по посадочным местам - до 50 мест.
- в Москве есть 524 улицы на которых расположилось только одно заведение общественного питания. 

Если же говорить о рекомендациях по поводу открытия кафе, то я вижу их такими:
- для определения с форматом лучше ориентироваться на тройку лидеров: кафе, столовая, ресторан. Предоставленные данные не дают возможность понять трафик/доходы данных типов заведений. Но думаю можно утверждать, что популярность таких форматов (их много) косвенно подтверждает рентабельность и живучесть таких заведений.  
- рекомендуемое количество посадочных мест до 50. Это самый популярный формат как среди сетевых, так и несетевых заведений. 
- ЦАО, ЮЗАО, САО, ЮАО, ЗАО - это округа в которых есть самые насыщенные улицы по количеству заведений. Думаю, что надо искать локацию рядом с топовыми улицами, но не на самих топовых улицах. Логика такая:
    - топовые улицы уже известны жителям города и туристам. Они притягивают трафик (посетителей) без дополнительных инвестиций (сами по себе), но и конкуренция там выше. Скорее всего и аренда в данных локациях выше чем в других местах
    - если расположить заведение рядом с топовыми улицами, то возможно удастся привлекать к себе часть посетителей от «топовых» маршрутов 
    - также, как мне кажется, стоит обогатить данные информацией о бизнес-центрах города. Думаю, что если наложить карту кафе на карту БЦ, то можно найти потенциально интересные места для открытия. В таком случае можно будет рассчитывать на поток посетителей из БЦ. 

Что касается возможности развития сети. Честно говоря не очень понятен вопрос и не понимаю, что именно здесь надо комментировать.
